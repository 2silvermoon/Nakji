## 개요 ##
학생들은 평일 야간 및 주말, 휴일에 학교에 출입하려면 출입 인증이 된 학생증이 필요하다.

하지만 일부 학생들은 학생증을 안만들었거나 출입 인증이 안된 학생증을 가졌고 또는 아예 들고 다니지를 않는다.

학생증은 안들고다녀도 휴대폰은 들고 다니니까 학생증 기능을 하는 것을 만들고자 한다.
